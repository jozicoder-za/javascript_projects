# 🐍 PythonPath AI – Learning Journey Dashboard

## Description

PythonPath AI is an AI-assisted learning dashboard designed to help JavaScript developers learn Python through a structured, interactive learning journey.

The application demonstrates the **"Using AI to Learn New Skills"** prompt template by combining a guided learning roadmap with AI-generated learning prompts, verification challenges, progress tracking, and personal learning reflections.

Instead of simply listing Python topics, PythonPath AI provides a structured learning experience where learners progress through carefully designed phases, use AI to deepen their understanding, complete practical verification projects, and monitor their progress through an interactive dashboard.

This project was developed as part of the **WeThinkCode Generative AI Course**.

# ✨ Key Features

## 📚 Structured Learning Journey

The application follows a four-phase roadmap:

### Phase 1 – Language Fundamentals

- JavaScript vs Python Syntax
- Static vs Dynamic Typing
- Variables and Primitive Types
- Control Structures
- Phase 1 Verification Project

### Phase 2 – Object-Oriented Programming

- Classes
- Constructors
- Methods
- Inheritance
- Phase 2 Verification Project

### Phase 3 – Python Features

- Lists vs Arrays
- Dictionaries
- Exception Handling
- List Comprehensions
- Lambda Functions
- Phase 3 Verification Project

### Phase 4 – Python Ecosystem

- Virtual Environments
- pip
- Pytest
- Python Standard Library
- Phase 4 Verification Project

## 🤖 AI Learning Prompts

Each lesson automatically generates an AI learning prompt based on the following prompt engineering template:

> "I'm beginning to learn Python coming from JavaScript. Before diving in, could you:
>
> 1. List the key technical terms I should know in Python that don't exist in JavaScript.
> 2. Provide a brief explanation of each term.
> 3. Show how these terms relate to each other.
> 4. Compare the concepts with JavaScript.
> 5. Explain common mistakes JavaScript developers make.
> 6. Provide beginner examples.
> 7. Create a practical coding exercise."

# ▶️ Usage

1. Open the application in your web browser.
2. Browse the Learning Journey roadmap.
3. Select a lesson.
4. Click **Learn**.
5. Copy the generated AI prompt.
6. Paste the prompt into ChatGPT (or another AI assistant).
7. Study the lesson.
8. Complete the verification challenge.
9. Mark the lesson as complete.
10. Record your notes and reflection.

# ❗ Troubleshooting

## Progress is not saving

Ensure Local Storage is enabled in your browser.

---

## Dark Mode resets

Refresh the browser after changing themes. Theme preferences are stored automatically.

---

## AI Prompt is empty

Click the **Learn** button on any lesson before attempting to copy the prompt.
