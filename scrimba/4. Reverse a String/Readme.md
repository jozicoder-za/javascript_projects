## Reverse a String

**_ Reverse the Provided String _**

You may need to turn the string into an array before you can reverse it
Your result must be a string

**_ Example _**

- reverseAString('hello') return 'olleh'

- reverseAString('Howdy') return 'ydwoH'

**_ Hints _**

- join()

- split()

- reverse()
